url规则：

1.path 是绝对路径的形式： path = “/apath”，baseUrl = “http://host:port/a/b” Url = “http://host:port/apath” 2.path 是相对路径，baseUrl 是目录形式： path = “apath”，baseUrl = “http://host:port/a/b/” Url = “http://host:port/a/b/apath” > * path 是相对路径，baseUrl 是文件形式： path = “apath”，baseUrl = “http://host:port/a/b” Url = “http://host:port/a/apath” > * path 是完整的 Url： path = “http://host:port/aa/apath“，baseUrl = “http://host:port/a/b” Url = “http://host:port/aa/apath”
